# OpenWiki

> 你的智能个人知识库

OpenWiki 是一款由人工智能驱动的个人知识管理工具。它可以将你收集的文章链接和文档自动整理成结构化的 Wiki 知识库，让你轻松存储、搜索和探索知识。

---

## ✨ 核心功能

### 📥 一键导入内容
- 粘贴任意文章链接，系统自动抓取内容
- 支持上传 Markdown 和纯文本文件
- 后台自动处理，无需手动整理

### 🔮 AI 智能编译
- 自动提取关键概念和重要标签
- 生成结构清晰的 Wiki 条目
- 发现内容之间的隐性关联

### 🔍 语义搜索
- 不用记住关键词，用自然语言描述你想找的内容
- AI 理解你的意图，找到最相关的知识

### 💬 智能问答
- 向知识库提问，获得精准回答
- 每个回答都附带原文出处，可溯源验证

### 📦 随时导出
- 一键下载整个知识库为 Markdown 文件
- 数据随时可带走，无锁定

## 🚀 快速开始

### 第一步：启动服务

确保已安装 Docker，然后运行：

```bash
docker compose up -d
```

### 第二步：配置 API 密钥

复制配置文件：

```bash
cp backend/.env.example backend/.env
```

编辑 `backend/.env`，填入你的 Claude API 密钥：

```bash
ANTHROPIC_API_KEY=sk-ant-...
```

> 还没有密钥？前往 https://console.anthropic.com/ 申请

### 第三步：初始化数据库

```bash
cd backend
uv sync
uv run python -m app.init_db
```

### 第四步：启动系统

在 `backend` 目录下：

```bash
uv run uvicorn app.main:app --reload --port 8000
```

在 `frontend` 目录下（新开一个终端）：

```bash
cd frontend
npm install
npm run dev
```

### 第五步：开始使用

打开浏览器访问 http://localhost:3000

## 📖 使用指南

### 导入知识

1. 点击导航栏的 **Inbox**（收件箱）
2. 粘贴文章链接或拖入文件
3. 系统会自动处理并编译为 Wiki 条目
4. 在 **Wiki** 页面查看整理好的内容

### 提问探索

1. 进入 **Ask**（问答）页面
2. 用自然语言提问，例如：
   - "什么是向量数据库？"
   - "总结 Rust 内存管理的特点"
3. 查看带引用的回答，点击引用可跳转到原文

### 导出数据

点击导航栏的 **Export** 按钮，所有 Wiki 条目将打包成 ZIP 文件下载。

## 🏗️ 系统架构

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   前端      │────▶│   后端      │────▶│  PostgreSQL │
│  (Next.js)  │     │  (FastAPI)  │     │  向量数据库   │
└─────────────┘     └──────┬──────┘     └─────────────┘
                           │
                    ┌──────┴──────┐
                    │             │
              ┌─────┴─────┐ ┌─────┴─────┐
              │   MinIO   │ │   Claude  │
              │  文件存储  │ │   AI 引擎  │
              └───────────┘ └───────────┘
```

数据流转：
1. **导入** - URL/文件 → 原始内容存储
2. **编译** - AI 提取概念、生成 Wiki 条目、创建向量索引
3. **查询** - 问题 → 语义匹配 → 智能回答
